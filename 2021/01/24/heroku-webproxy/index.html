免费容器 Heroku 搭建 web 代理

发表于 2021-01-24 更新于 2021-08-17 分类于 爬墙 阅读次数： 本文字数： 407 阅读时长 ≈ 1 分钟

说明：

  • Heroku 为我们提供了免费的容器服务，我们不应该滥用它，所以不宜做为长期翻墙使用。
  • Heroku 免费版有所限制，比如：30分钟无人访问就休眠以及所有实例访问总时长限制为550小时。

正文：(Heroku 注册过程就不赘述了)

  1. 点击下方链接进入 HolyUnblockerPublic 项目地址。https://github.com/QuiteAFancyEmerald/HolyUnblockerPublic
  2. 点击 Deploy to Heroku ，进入 Heroku 后台进行部署。
  3. 只需填入实例名字和选择实例地区即可。
  4. 点击下方的 Deploy app ，等待自动部署完毕。
  5. 点击 view ，会跳到已部署好的实例网站，如下图显示即为部署成功。

备注：

由于搭建的是 web 代理，使用起来可能会有些问题，所以不建议大流量使用，比如看视频、下东西等等。建议平时用来搜索点东西就好了。
