免费容器 IBM Cloud 搭建 Xray 代理

发表于 2021-01-26 更新于 2021-08-17 分类于 爬墙 阅读次数： 本文字数： 1.3k 阅读时长 ≈ 2 分钟

说明：

  1. IBM Cloud 为我们提供了免费的容器服务，我们不应该滥用它，所以不宜做为长期翻墙使用。
  2. IBM Cloud 免费版有所限制，几天无人访问就会休眠。（具体是几天已经忘记了，不过下面讲述的方法会每天定时重启实例，所以无需担心）

正文：(IBM Cloud 注册过程就不赘述了，注册不成功，就换邮箱、IP)

  • 点击下方链接进入 xray-ibm-us 项目地址。（此项目仅适用于达拉斯地区，伦敦地区的请看下方备注）
    https://github.com/zyp997/xray-ibm-us
  • 点击右上角 Fork ，会跳转到你的项目地址。（需要 GitHub 账号）
  • 点击 Settings ，在左边栏目找到并点击 Secrets ，再点击右上角的 New repository secret，分别创建如下表的参数。
Secrets变量说明
IBM_CF_USERNAMEIBM Cloud 邮箱地址
IBM_CF_PASSWORDIBM Cloud 邮箱密码
IBM_CF_APP_NAMEIBM Cloud 应用程序名
V2_UUID自定义 UUID 码
V2_WS_PATH_VMESS V2_WS_PATH_VLESS左边协议选择一个，填入自定义 PATH 路径
  • 点击 Actions ，点击 I understand my workflows, go and ahead and enable them ，在左边栏目找到并点击 IBM Cloud Foundry - Xray ，点击 Enable workflow ，再点击 Run workflow 选项，最后点击 Run workflow 。
  • 在左边栏目找到并点击 All workflows ，如出现下图结果则表示实例部署成功。(如果什么都没显示，则在 README.md 文件随便打个字符即可；如果执行失败，则仔细检查 Secrets 变量 是否填写正确，之后再重新执行 Actions)
  • 登录 IBM Cloud，进入到部署好的 Cloud Foundry 应用程序，点击右上角的 操作… 选项，点击 编辑路径，修改为 IBM Cloud 应用程序名.us-south.cf.appdomain.cloud。点击 保存，选中实例，点击 除去。
  • 下载支持 vless 和 vmess 协议的客户端，配置信息如下：

代理协议：vless 或 vmess
地址：IBM Cloud 应用程序名.us-south.cf.appdomain.cloud
端口：443
UUID：自定义 UUID 码
加密方式：none
传输协议：ws
路径：自定义PATH路径
底层传输安全：tls

备注：

伦敦地区的，需到自己 fork 的项目下，编辑 .github/workflows 目录下的 ibm-cf-v2.yml 文件，将 env 项下的 IBM_CF_API 参数修改为：https://api.eu-gb.cf.cloud.ibm.com。

由于 IBM Cloud 给的免费内存空间为256MB，而项目中的内存设置的为128MB，所以可以多搭建一个来玩玩。

还有使用 v2ray 进行部署的项目，配置信息如上所述，有兴趣的可以到下面的地址进行部署。
https://github.com/zyp997/v2ray-ibm-us
