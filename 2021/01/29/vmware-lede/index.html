使用 VMware 虚拟机安装 Lede 软路由

发表于 2021-01-29 更新于 2021-08-17 分类于 实用 阅读次数： 本文字数： 996 阅读时长 ≈ 2 分钟

说明：

  1. Lede 是路由器固件项目 OpenWRT 的一个复刻分支项目，并继承原来 OpenWRT 的开发目标。
  2. 软路由的可玩性很高，功能也挺多的。而其中 Lede 软路由的酷软中心可提供诸多的插件，因而备受网友喜爱。

正文：（注意：没有提及的部分默认选择下一步）

  • 下载 VMware 虚拟机安装包进行安装。
  • 点击下方链接进入网站，下载后缀为 vmdk 的磁盘文件。https://firmware.koolshare.cn/LEDE_X64_fw867/%E8%99%9A%E6%8B%9F%E6%9C%BA%E8%BD%AC%E7%9B%98%E6%88%96PE%E4%B8%8B%E5%86%99%E7%9B%98%E4%B8%93%E7%94%A8/
  • 打开 VMware 虚拟机，点击 创建新的虚拟机，选择 自定义。
  • 选择 稍后安装操作系统。
  • 选择 Linux，选择 其他Linux或更高版本内核64位。
  • 虚拟机名称和位置随意填写和选择。（注意：安装位置至少留出1GB的存储空间）
  • 虚拟机内存选择1GB。选择 使用现有虚拟磁盘。
  • 创建完虚拟机后，点击 编辑虚拟机设置，点击 网络适配器，选择 桥接模式，并勾选 复制物理网络连接状态。点击 硬盘，点击 拓展，修改磁盘大小为1GB。
  • 点击 开启此虚拟机，画面停止滚动后，按回车。
  • 输入下面代码修改网络配置文件。
  • 按键盘的 i 键，修改 config interface ‘lan’ 下的 option ipaddr 项的 IP 地址。（更改为物理机 IP 段内任意空闲 IP 地址。不知道物理机的 IP 段可到 控制面版 中，点击 网络和共享中心 或是 查看网络状态和任务，点击正在连接的网络，点击 详细信息，查看 IPv4 地址 的网段和 IPv4 默认网关 并记录下来，例如：IPv4 地址为 192.168.1.1，则修改的地址可以为 192.168.1.123）
  • 按 ESC 键后输入 :wq 保存并退出。
  • 输入 reboot 进行重启。
  • 打开浏览器，输入刚才修改的 IP 地址。（我的是 192.168.1.123）
  • 输入默认密码 koolshare，左侧列表点击 网络-接口，点击 LAN 口的 编辑。
  • IPv4 网关 填写先前记录的 IPv4默认网关，使用自定义的DNS服务器 可填写为223.5.5.5 和 223.6.6.6，点击 保存并应用。
