宝塔面板简单优化

发表于 2021-01-30 更新于 2021-08-11 分类于 建站 阅读次数： 本文字数： 1k 阅读时长 ≈ 2 分钟

说明：

安装完宝塔面板并且建好网站之后，需要给面板和服务器做一些简单的设置，比如说 php 优化、myql 优化、开启 php 缓存、网站流量限制等等来提高服务器的性能。

正文：

定期释放内存（宝塔面板-计划任务-释放内存）

释放 PHP、MYSQL、PURE-FTPD、APACHE、NGINX 的内存占用,建议在每天半夜执行！

开启 Swap/虚拟内存（宝塔面板- Linux 工具箱- Swap/虚拟内存）

官方已经给出了推荐的 Swap 的设置值。

物理内存 Swap
512 MB 1024 MB
1 GB 1500 MB
2 GB 2048 MB
4 GB 4096 MB
6 GB+ 4096 MB

安装 PHP 缓存扩展（宝塔面板-软件商店-PHP 设置-安装拓展）

php 版本强烈建议选择 php7.0 以上版本的， php 缓存扩展推荐使用 Opcache 脚本缓存和 Memcached 或者 redis 内容缓存。

PHP 配置调整（宝塔面板-软件商店-PHP 设置-配置修改）

Wordpress 要跑的流畅的话还是很耗内存的，Wordpress 本身就是个吃内存的大户，所以我们需要在 php 管理的配置中，把 memory_limit （脚本内存限制）修改成 256M。

PHP 性能调整（宝塔面板-软件商店-PHP 设置-性能调整）

里面自带了几个优化过的 php 并发方案，根据自己 VPS 的内存进行选择。

开启隐藏的 waf 防火墙（宝塔面板-软件商店-Nginx 设置-配置修改）

找到大约在第17行的代码：**#include luawaf.conf;**，去掉前面的 **#** 符号，保存并重启 Nginx。

访问 **http://你的网址/?id=../etc/passwd**，页面会弹出拦截提示，如下图显示，则成功开启了 ngx_lua_waf。

网站流量限制（宝塔面板-网站 设置-流量限制）

里面自带了几个优化过的限制方案，我们一般选择 论坛/博客 这种就可以了，这样对网站有个最起码的保护作用，或者说如果被 cc 了，这样设置还是有点作用的。

MySQL 性能调整（宝塔面板-软件商店- MySQL 设置-性能调整）

里面自带了几个优化过的方案，根据自己 VPS 的内存进行选择。当你有在使用 redis、memcached 等缓存软件时，将 query_cache_size 设为0可以将其关闭。

备注：

这里提供的教程也只是一些最简单最基本的设置，对一些小白可以能会有很大帮助，如果你们有更好的优化方案，不妨分享出来。

如有任何疑问，可在下方评论区@我或者发送悄悄话都可以，我会尽力回答。
