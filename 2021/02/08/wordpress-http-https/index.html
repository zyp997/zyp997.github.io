WordPress 网站设置 HTTP 协议强制跳转到 HTTPS 协议

发表于 2021-02-08 更新于 2021-08-17 分类于 建站 阅读次数： 本文字数： 3k 阅读时长 ≈ 5 分钟

说明：

在互联网火热的今天，安全问题显得越来越重要，为了用户信息安全，很多热门网站都启用了 https。有小伙伴就问：我启用了 https，为什么访问的时候显示的还是 http 呢？其实，有时候并不是因为我们 ssl 证书配置的不对，而是因为我们没有做 https 强制跳转。

正文：

  • 在 宝塔面板-网站-设置-SSL，点击右上角的 强制HTTPS。
  • 访问网站，如果还显示连接不安全，则说明还有资源文件走的是 http，还需要做下面两种操作的其中一种。
