为 blog 制作一个全屏的视频主页

发表于 2021-03-16 更新于 2021-08-11 分类于 建站 阅读次数： 本文字数： 4k 阅读时长 ≈ 7 分钟

说明：

之前看到过别人用全屏的 gifi 动图做主页，然后下滑就能显示博客的文章。不过这样并不合我的喜好，我更喜欢全屏的视频页，提示点击跳转到博客文章页。
一开始是想着网上应该有模板或者插件可以实现，找了挺久的，找了个半成品的模板拿来修改。改是改成功了，不过有个问题就是：移动端的无法播放视频。在网上找解决方法和自己尝试修改也没成功，最终还是自己写吧。于是便自己做了一个全屏的视频主页。

正文：

制作 php 模板

重点：导入 font-awesome.css 文件以使用 Font Awesome 图标，video 标签属性设置为 autoplay(自动播放)、loop muted(静音循环播放)，创建一个标签绑定 onclick 事件(点击标签跳转到博客文章页)。

制作 css 样式

  • 实现视频铺满整个页面：设置所有的元素的对齐方式以及和父类之间的间距都为0，设置 video 元素为 fixed 的绝对定位以及最小的宽度和高度为100%。
  • 实现圆形图标居中：设置 img 元素为 absolute 的绝对定位以及圆角为50%、单单设置 top 和 left 属性为50%，会使图标偏右下，还需要补充设置 margin-top 和 margin-left 属性为图片宽高的一半。
  • 左右下角的标签样式就是细节的问题，就不多说了，自己百度一下或者改一下代码就能看出每个属性的具体作用了。

制作 js 脚本

  • 导入 jquery.min.js 文件，否则会无法正常显示和实现功能。
  • 实现图片不停旋转、鼠标悬停停止旋转：设置定时器（旋转方向、速度和时间），通过绑定 onmousemove 和 onmouseleave 事件给 img 标签实现鼠标悬浮在图片上时清除定时器，离开图片时继续运行定时器。
  • 实现点击视频开关声音：绑定 onclick 事件给 video 标签，点击视频时反转 video 标签的 muted 属性实现开关声音。

设置 wordpress 主页为制作好的模板

  • 将制作好的模板文件放到当前主题目录下。（记得修改图片和视频路径）
  • 进入 wordpress 后台，新建一个页面，填写标题为 主页，在设置的页面属性中，找到 **模板:**，选择 主页。再新建一个页面，填写标题为 文章，在设置的页面属性中，找到 **模板:**，选择 默认模板。
  • wordpress 后台中点击 外观-自定义，点击 主页设置，如下图选择，最后保存更改即可。

备注：

想要完整的页面模板，请到我的网盘下载 视频主页模板 压缩包。(想要更多视频可到我的 网盘-生活 目录下载)
网盘里的视频是我已经压缩好了的，如果想用自己的视频尽量压缩之后再使用。
