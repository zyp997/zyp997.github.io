使用 WP Rocket 插件加速 blog

发表于 2021-03-26 更新于 2021-08-11 分类于 建站 阅读次数： Valine： 本文字数： 819 阅读时长 ≈ 1 分钟

说明：

WP Rocket 是 WordPress 很流行的付费缓存插件。WP Rocket 不仅仅是一个简单的缓存插件，它有许多额外功能。提供了许多自定义设置，例如文件优化，预加载，延迟加载，数据库优化等等。

请到我的网盘 建站 目录下载 wp-rocket 插件破解版 压缩包。

仪表盘配置

不需要动任何东西。

缓存配置

如下图配置。如果网站有多个用户，就勾选 为已登录的 WordPress 用户启用缓存，如果只有你自己就不要勾选。缓存有效期 默认10小时，如果网站更新频率快可以改短些，否则默认即可。

文件优化配置

建议全部勾选后测试，没问题就最好了，可以起到最大程度的加速效果。如果有异常就取消某项勾选，保存后再测试。大部分的问题出在 压缩 JS 文件 选项，也有部分网站问题出现在 合并CSS文件、合并 JavaScript 文件 选项。

媒体配置

如下图配置。懒加载通俗地说就是页面滚动到哪里，媒体才开始加载。下面没出来的就不加载，这样可以提高网页打开速度。懒加载不是每个主题都合适的，也有可能起反作用，所以需要视个人情况配置。是否勾选 开启 WebP 缓存，就看你是否有使用 WebP 图片。

预缓存配置

如下图配置。预缓存就是提前生成缓存文件，这样可以提高网页打开速度。

数据库

如下图所示，就是简单的文章、评论、临时数据清理，如果有需要还可以优化数据库表和设置定时清理任务。注意：对数据库的操作建议进行备份之后再进行。

CDN 配置

所有静态文件 URL（CSS、JS和图片）将被重写至下方您提供的 CNAME。由于我使用了 Cloudflare，有额外的拓展给我使用，下面会讲到。
