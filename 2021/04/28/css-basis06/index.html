父级边框塌陷解决方法：

  • 增加父元素的宽或高
    缺点：简单，但如果父元素设有了固定的高度，就会被限制。
  • 增加一个空的 div 标签，并设置 style 的 clear 属性为 both。
    缺点：简单，但代码中尽量避免空 div 标签。
  • 设置父元素 style 的 overflow 属性
    缺点：简单，但子元素可能无法显示完全，并且下拉的一些场景尽量避免使用。
  • 父元素添加一个 after 伪类
    缺点：写法稍微复杂一点，但是没有副作用，推荐使用！

display 和 float 属性对比

共同点：都能实现元素在同一行进行排列。
不同点：display 属性不能控制方向，float 属性可以。使用 float 属性时可能会出现父级边框塌陷问题，display 属性则不会。
