DQL（最重点）

准备工作

因为要学习查询语句，得先把数据存入。记得把之前创建的数据库都删掉。

select 语法

[ ] 里的内容表示可选的，{ } 里的内容表示必选的。注意：关键字之间都是有固定顺序的。

指定查询字段

AS（as 关键字可省略）

  • 给数据列取一个新别名
  • 给数据库表取一个新别名
  • 给经计算或总结的结果取一个新别名

DISTINCT

去掉 SELECT 查询返回的记录结果中重复的记录（列的值相同的记录），只返回一条。

使用表达式的列

数据库中的表达式：一般由文本值、列值、NULL、函数和操作符等组成。

应用场景 :

  • SELECT 语句返回结果在列中使用
  • SELECT 语句中的 ORDER BY、HAVING 等子句中使用
  • DML 语句中的 WHERE 条件语句中使用表达式
