Npm 命令解释：

  • npm install 模块名：安装模块到项目目录下
  • npm install -g 模块名：-g 的意思是将模块安装到全局
  • npm install -save 模块名：–save 的意思是将模块安装到项目目录下， 并在 package.json 文件的 dependencies 节点写入依赖，-S 为该命令的缩写。
  • npm install -save-dev 模块名：–save-dev 的意思是将模块安装到项目目录下，并在 package.json 文件的 devDependencies 节点写入依赖，-D 为该命令的缩写。
