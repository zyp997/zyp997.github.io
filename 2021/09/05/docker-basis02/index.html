常用容器命令

docker run

用于新建并启动容器，语法如下：

参数：

  • --name 容器名：容器取名
  • -d：后台运行
  • -it /bin/控制台(bash/sh…)：使用交互方式运行，并进入容器。-it 就是为该 docker 创建一个伪终端，这样就可以进入到容器的交互模式，也就是进入到容器里面。/bin/bash 的意思是表示载入容器后运行 bash，容器中必须要保持一个进程的运行，这样做就是为了防止整个容器启动后马上杀死自己。
  • -p：容器端口映射
  • -p ip:宿主机端口:容器端口
  • -p 宿主机端口:容器端口（常用）
  • -p 容器端口
