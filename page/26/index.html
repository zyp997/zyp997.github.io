数据库分类

  • 关系型数据库 (SQL)
    • MySQL , Oracle , SQL Server …
    • 关系型数据库通过外键关联来建立表与表之间的关系。
  • 非关系型数据库 (NOSQL - Not Only SQL)
    • Redis , MongoDB …
    • 非关系型数据库通常指数据以对象的形式存储在数据库中，而对象之间的关系通过每个对象自身的属性来决定。
