搭建站点监控网站

发表于 2021-03-08 更新于 2021-08-17 分类于 建站 本文字数： 2.7k 阅读时长 ≈ 5 分钟

说明：

  1. uptime-status - 一个基于 UptimeRobot API 的在线状态面板
  2. UptimeRobot 是一家能够提供定时监控服务的网站。比如有些站长有多台服务器和多个网站，那么你需要时刻都关心你的 vps 或者网站的运行情况。如果遇到宕机，或者网站打不开都能第一时间得到通知。
  3. 免费套餐提供了50个监控项目，2个月的日志存储，没有 SMS 短信服务。对于大多数站长应该是已经完全够用了。

正文：

  • 点击下方链接注册账号，默认的是免费套餐。
    https://uptimerobot.com/signUp
  • 登录后会进入后台，点击 Add New Monitor，选择相应的监控类型，有如下的选项。

HTTP(s)：用于检测 HTTP 以及 HTTPS 网站，网站无法访问则邮件通知。
Keyword：用于检测某网站是否出现某关键词，可以设置为“出现 xxx 关键词”或者“关键词 xxx 消失”则邮件通知我。
Ping：用于检测服务器是否正常开机运行，ping不通则邮件通知，不能用于检测某 VPS 是否被 q。
Port：用于检测服务器某指定端口是否开放，若端口关闭则邮件通知。
